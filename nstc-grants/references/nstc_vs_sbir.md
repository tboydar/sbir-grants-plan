# 國科會專題研究計畫 vs SBIR 完整差異比較

> **重要聲明**：本文件整合國科會及經濟部官方資訊，供研究人員與企業了解兩類計畫差異，選擇最適合的申請路徑。確切規定以各主管機關最新公告為準。

---

## 一、核心差異一覽

| 比較項目 | 國科會專題研究計畫 | SBIR 小型企業創新研發計畫 |
|---------|-------------------|-------------------------|
| **主管機關** | 國家科學及技術委員會（國科會/NSTC） | 經濟部中小及新創企業署 |
| **核心目的** | 推動學術研究、培育研究人才 | 鼓勵中小企業技術創新與商品化 |
| **申請主體** | 大學/研究機構之教師或研究人員 | 依法登記之中小企業 |
| **計畫主持人** | 大學教授/研究人員 | 企業負責人或研發主管 |
| **審查重點** | 學術原創性、研究方法嚴謹性 | 技術創新性、商業化可行性 |
| **成果導向** | 學術論文、專利、人才培育 | 商品化、市場營收、就業創造 |
| **補助性質** | 全額補助（無需自籌款） | 補助不超過計畫總經費 50%（需自籌） |
| **執行場域** | 大學/研究機構實驗室 | 企業內部研發場域 |

---

## 二、申請資格比較

### 國科會專題研究計畫

- 公私立大專院校**助理教授以上**專任人員
- 公私立研究機構**副研究員以上**（或具博士學位之助理研究員以上）
- 醫療機構之**主治醫師滿 2 年**或碩士學位從事研究滿 3 年者
- 講師職滿 3 年且有學術著作發表者
- 需由所屬**機構**函送申請

### SBIR 計畫

- 依法完成公司或商業登記之**中小企業**
- 實收資本額 ≤ NT$1 億或員工 < 200 人
- 5 年內無重大政府計畫違約紀錄
- 3 年內無欠稅情事
- 非陸資企業或外國分公司
- **補助款 ≤ 自籌款 ≤ 實收資本額**

## 三、補助金額與期程比較

### 國科會專題研究計畫

| 計畫類型 | 年度金額範圍 | 執行期限 |
|----------|-------------|----------|
| 個別型專題 | NT$50 萬 - 200 萬/年（依學門） | 1-5 年 |
| 新進人員 | NT$50 萬 - 100 萬/年 | 1-3 年 |
| 整合型 | NT$500 萬 - 2,000 萬/年（整體） | 通常 3 年 |
| 產學合作 | NT$100 萬 - 500 萬/年 | 1-3 年 |
| 優秀年輕學者 | 每年不超過 NT$500 萬 | 至多 4 年 |
| 青穗學者 | 每年不超過 NT$800 萬 | 至多 4 年（可續 2 期共 8 年） |
| 大專學生 | 研究津貼 NT$6,000/月 + 耗材 NT$20,000 | 8 個月 |

### SBIR 計畫（115 年度/2026 年起）

| 階段 | 補助上限（個別申請） | 執行期程 |
|------|---------------------|----------|
| Phase 1（先期研究） | NT$150 萬 | 6 個月 |
| Phase 2（研究開發） | 每年 NT$600 萬，全程上限 NT$1,200 萬 | 6 個月 - 2 年 |
| Phase 2+（加值應用） | NT$600 萬 | 6 個月 - 1 年 |
| 聯盟 Phase 2 | 家數 x NT$500 萬，上限 NT$3,000 萬 | 9 個月 - 2 年 |

**關鍵差異**：國科會為全額補助；SBIR 補助不超過 50%，企業需自籌對等款項。

## 四、審查標準比較

### 國科會審查重點

1. **計畫主持人研究表現與執行計畫能力**（學術 track record）
2. **計畫主題之重要性與創新性**（學術突破性）
3. **研究內容與方法之可行性**（方法論嚴謹性）
4. **預期完成之項目與成果**（學術貢獻）
5. **經費與人力之合理性**

### SBIR 審查重點

1. **技術創新性**：是否為新技術、新產品或新服務
2. **市場可行性**：是否有明確市場需求與商業模式
3. **執行可行性**：技術路線與時程是否合理
4. **團隊能力**：企業是否具備執行研發的能力
5. **經費合理性**：預算編列是否符合研發內容

### 本質差異

| 面向 | 國科會 | SBIR |
|------|--------|------|
| **成功標準** | 學術突破、論文發表 | 產品商品化、營收增長 |
| **風險容忍** | 允許「失敗但有學術價值」的探索 | 期待具體可商品化的成果 |
| **國際比較** | 是否達國際學術前沿 | 是否具國際市場競爭力 |
| **人才培育** | 培養研究生、博士後 | 提升企業研發人力 |

## 五、經費編列比較

### 國科會經費科目

| 科目 | 內容 | 比例限制 |
|------|------|----------|
| 業務費-人力費 | 專/兼任助理、主持費 | 無固定上限，依需求 |
| 業務費-耗材雜支 | 耗材、物品、圖書 | 核實列支 |
| 研究設備費 | 單價 ≥ 1 萬且耐用 ≥ 2 年 | 依核定清單 |
| 國外差旅費 | 國際合作、學術會議 | 依核定種類 |
| 管理費 | 學校統籌支用 | 依國科會核定比率 |

### SBIR 經費科目

| 科目 | 內容 | 比例限制 |
|------|------|----------|
| 人事費 | 研發人員薪資、保險 | 建議 ≤ 60% |
| 消耗性器材及原材料 | 研發耗材 | 建議 ≤ 25% |
| 研發設備使用費 | 設備折舊或租賃 | 無固定上限 |
| 技術移轉費 | 技術引進或專利購買 | 整體 ≤ 60%；專利購買 ≤ 30% |
| 國內差旅費 | 計畫相關差旅 | 無固定上限 |
| 專利申請費（115 年新增） | 國內外專利申請 | 國內 3 萬/案、國外 10 萬/案 |

## 六、申請時程比較

### 國科會

| 時間 | 事項 |
|------|------|
| 每年 11 月初 | 年度大批計畫線上申請開始 |
| 每年 12 月底-1 月初 | 大批計畫校內截止（各校不同） |
| 隨到隨審 | 新進人員計畫、特殊身分申請 |
| 不定期 | 國際合作計畫（依合作國時程） |
| 每年 3-4 月 | 產學合作計畫徵求 |
| 每年 2-3 月 | 大專學生研究計畫申請 |

### SBIR

| 類型 | 時間 |
|------|------|
| 中央型 SBIR | **隨到隨審**，全年可送件 |
| 地方型 SBIR | 依各縣市政府公告，通常每年開放 1 個月 |
| 送件到審查結果 | 約 2-3 個月 |

## 七、成果報告與後續義務

### 國科會

- 計畫執行期滿後 **3 個月內**繳交電子檔成果報告
- 成果報告應立即公開（涉及專利/論文者可延後公開，最長 2 年）
- 涉及人體研究須繳交「研究中的性別考量報告表」
- 出國者須繳交心得報告、論文全文或摘要
- 未依規定繳交成果報告者，國科會**不再核給**專題研究計畫

### SBIR

- 依合約規定繳交期中及期末報告
- Phase 2+ 結案後 3 年內須取得研發成果商品訂單
- 需繳交訂單金額 1% 作為研發回饋金（上限為補助款 2%）
- 未完成回饋者，面臨 3 年內不得申請 SBIR 的停權

## 八、智慧財產權歸屬

| 面向 | 國科會 | SBIR |
|------|--------|------|
| **IP 歸屬** | 通常歸執行機構（大學） | 歸申請企業 |
| **技轉義務** | 鼓勵但非強制技轉 | 以商品化為目標 |
| **論文發表** | 鼓勵且為重要績效 | 非主要要求 |
| **專利布局** | 鼓勵但非必要 | 高度鼓勵（115 年新增專利費補助） |

## 九、如何選擇申請路徑

### 適合申請國科會專題研究計畫的情境

- 您是大學教授或研究機構研究員
- 研究目的以**學術探索**和**知識創造**為主
- 需要培育研究生和博士後
- 研究成果主要以論文形式呈現
- 需要長期穩定的研究支持
- 想建立個人學術聲譽與研究能量

### 適合申請 SBIR 的情境

- 您是中小企業負責人或研發主管
- 已有明確的**產品/技術/服務創新**構想
- 目標是開發可上市的**商品化成果**
- 企業有足夠**自籌款**能力（至少等於補助款）
- 希望快速將技術推向市場

### 兩者可銜接的情境

- **學術研究 → 產學合作 → SBIR**：教授的研究成果先透過國科會產學合作計畫與企業合作，驗證可行性後，由企業以 SBIR 進行商品化
- **SBIR → 國科會產學合作**：企業在 SBIR 過程中發現需要更深入的基礎研究，與大學教授合作申請產學合作計畫
- 國科會「研究計畫產學加值鼓勵方案（ARRIVE 方案）」可作為學術成果走向產業的橋樑

## 十、常見誤解澄清

| 誤解 | 正確說明 |
|------|----------|
| 「國科會計畫也可以由企業申請」 | 國科會專題計畫申請主體為大學/研究機構，企業不得直接申請 |
| 「SBIR 可以找教授當主持人」 | SBIR 計畫主持人必須是企業人員，教授僅能擔任技術顧問或委外合作單位 |
| 「國科會計畫不看產業應用」 | 產學合作計畫要求有企業配合款及應用價值 |
| 「SBIR 不需要技術深度」 | SBIR 審查非常重視技術創新性與可行性 |
| 「兩個計畫的錢可以混用」 | 絕對不可，不同來源經費必須專款專用 |
| 「國科會計畫有自籌款要求」 | 國科會一般專題計畫為全額補助，無自籌款要求（產學合作除外） |
| 「SBIR 通過率很低」 | SBIR 通過率約 40-50%，與國科會相當 |

## 附錄：完整比較總表

| 比較維度 | 國科會專題研究計畫 | SBIR |
|---------|-------------------|------|
| 主管機關 | 國科會（NSTC） | 經濟部 |
| 法源依據 | 補助專題研究計畫作業要點 | 小型企業創新研發計畫推動作業計畫 |
| 申請主體 | 大學/研究機構人員 | 中小企業 |
| 自籌款 | 不需要（產學合作除外） | 需自籌 ≥ 50% |
| 最高補助 | 依計畫類型，青穗學者可達 800 萬/年 | Phase 2 個別 600 萬/年（115 年起） |
| 申請方式 | 學術研發服務網線上申請 | SBIR 線上申請系統 |
| 審查方式 | 同儕書面審查為主 | 書面 + 簡報審查 |
| 核定率 | 約 40-60% | 約 40-50% |
| 執行考核 | 期中進度報告 + 期末成果報告 | 期中 + 期末報告 + 商品化追蹤 |
| 結餘款 | 校務基金學校得免繳回 | 依合約辦理 |
| 核銷方式 | 依政府支出憑證處理要點 | 依 SBIR 會計科目及編列原則 |
| 研究倫理 | 需完成學術倫理訓練 | 無此要求 |
