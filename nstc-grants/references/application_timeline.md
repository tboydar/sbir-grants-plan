# 國科會專題研究計畫申請流程與時程

> **重要聲明**：本文件依據國科會官方公告及各大學研發處公告整理。確切日期每年微調，請以國科會最新公告及各校研發處通知為準。

---

## 一、年度大批計畫申請時程（最重要）

### 115 年度（2026 年）實際時程

| 日期 | 事項 |
|------|------|
| **114 年 11 月 4 日（2025/11/4）** | 國科會線上申請系統開放 |
| **114 年 12 月底 - 115 年 1 月初** | 各校校內截止日（各校不同） |
| **115 年 1 月 1-7 日左右** | 教師完成線上申請截止（依學校規定） |
| **115 年 1 月上旬** | 系所完成線上彙整、主持人資格審查 |
| **115 年 1 月上旬** | 研發處完成全校計畫彙整並函送國科會 |
| **115 年 2-5 月** | 國科會進行審查作業（6 個月內完成） |
| **115 年 6-7 月** | 審查結果公告 |
| **115 年 8 月 1 日** | 新年度計畫開始執行 |

### 一般年度流程（每年大致相同）

```
7-8月    國科會內部規劃下年度徵求
       ↓
10-11月  公告年度計畫徵求函
       ↓
11月初   線上申請系統開放
       ↓
12月底-1月初  各校校內截止
       ↓
1月上旬  學校函送國科會
       ↓
1-6月   審查作業期間
       ↓
6-7月   結果公告
       ↓
8月1日   新計畫執行起始日
```

## 二、各類計畫申請時程

### 2.1 新進人員研究計畫（隨到隨審）

| 項目 | 說明 |
|------|------|
| **申請時間** | 全年受理，隨到隨審 |
| **資格期限** | 從未申請國科會計畫者：起聘日/獲博士學位日/符合資格日起 **3 年內** |
| **注意事項** | 3 年內新進教師未曾申請者，應申請「新進人員隨到隨審」，勿與大批計畫一起申請 |
| **5 年內新進教師** | 曾申請國科會計畫者，應選「新進人員研究計畫（大批）」 |
| **申請方式** | 線上申請 + 校內公文系統創文 + 紙本送件 |

### 2.2 產學合作研究計畫

| 項目 | 說明 |
|------|------|
| **第一期** | 約每年 1 月中旬截止 |
| **第二期** | 約每年 5-6 月（依公告） |
| **特別要求** | 需有企業書面合作協議及配合款承諾 |

### 2.3 大專學生研究計畫

| 項目 | 說明 |
|------|------|
| **申請時間** | 約每年 12 月底 - 翌年 1 月（依公告） |
| **執行期間** | 約 7 月 - 翌年 2 月（8 個月） |
| **成果繳交** | 執行期滿後繳交研究報告 |

## 三、線上申請系統操作

### 3.1 學術研發服務網

- **網址**：[https://wsts.nstc.gov.tw/](https://wsts.nstc.gov.tw/)
- **功能**：帳號管理、計畫申請、進度報告、成果報告等
- **系統服務專線**：0800-212-058 或 (02) 2737-7592
- **Email**：misservice@nstc.gov.tw
- **上班時間**：週一至週五 8:30-17:30

### 3.2 申請前準備

| 準備事項 | 說明 |
|----------|------|
| 註冊帳號 | 至學術研發服務網註冊，確認身分為「研究人員」 |
| 更新現職 | 確認「現職/就讀學校」資訊正確（使用下拉選單選擇） |
| 學術倫理 | 首次申請者需完成 6 小時學術倫理教育訓練 |
| 個人資料表 | 更新近 5 年研究績效 |

### 3.3 線上申請步驟

1. 登入學術研發服務網
2. 進入「專題研究計畫線上申請系統」
3. 新增計畫，填寫 CM01 基本資料表
4. 線上填寫 CM02 中英文摘要及關鍵詞（已改為線上填寫，不需上傳）
5. 上傳計畫書 PDF（注意頁數限制）
6. 填寫 CM07 研究人力費表
7. 填寫設備費、國外差旅費等相關表格
8. 上傳必要附件（倫理審查文件、個人資料表等）
9. 確認所有資料完整後「送出」至系統
10. 由系所承辦人員線上確認
11. 研發處彙整後函送國科會

### 3.4 常見系統問題

| 問題 | 解決方式 |
|------|----------|
| 計畫狀態為「暫存」 | 至基本資料更新「現職/就讀學校」，使用下拉選單儲存後，在 CM01 按「同步現職」 |
| 查不到共同主持人 | 共同主持人須有學術研發服務網帳號且身分為「研究人員或延攬研究學者」 |
| 繳交截止日期不明 | 由申請機構行政人員依各計畫類別設定，請洽校內行政人員 |
| 計畫名稱修改 | 計畫狀態為暫存時，進入 CM01 基本資料表修改 |

## 四、申請必備文件清單

### 4.1 所有計畫共通文件

- [ ] 計畫申請書（CM01-CM07 等表格）
- [ ] 計畫主持人個人資料表
- [ ] 共同主持人個人資料表（如有）
- [ ] 研究績效或成果相關表格（近 5 年）
- [ ] 利益衝突聲明書（簽署）

### 4.2 首次申請者額外文件

- [ ] 6 小時學術倫理教育訓練證明
- [ ] 計畫主持人資格切結書（由學校出具）

### 4.3 涉及倫理審查的計畫

- [ ] 人體試驗：醫學倫理委員會核准文件
- [ ] 動物實驗：實驗動物管理委員會核准文件
- [ ] 基因重組：生物實驗安全委員會核准文件
- [ ] 人文社會涉及人的研究：研究倫理審查證明
- [ ] 人體研究：研究中的性別考量檢核表

### 4.4 產學合作計畫額外文件

- [ ] 企業書面合作協議
- [ ] 企業配合款承諾書
- [ ] 合作企業基本資料

### 4.5 國際合作計畫額外文件

- [ ] 合作國外研究人員 CV（依合作機構模板）
- [ ] 雙方合作說明書
- [ ] 對方機構確認函

## 五、計畫書頁數限制

| 計畫類型 | 頁數上限 |
|----------|----------|
| 個別型 1 年期 | 約 20 頁 |
| 個別型多年期 | 約 25 頁 |
| 整合型或單年經費 > NT$500 萬 | 約 40 頁 |
| 大專學生研究計畫 | 約 10 頁 |

> 實際頁數限制以各年度計畫徵求公告為準。

## 六、建議準備時程表

### 大批計畫（以 11 月開放申請為例）

| 時間點 | 工作事項 |
|--------|----------|
| **T-12 週（8 月）** | 確認研究方向、開始文獻蒐集 |
| **T-8 週（9 月）** | 確認計畫類型、開始撰寫計畫書初稿 |
| **T-6 週（10 月初）** | 完成計畫書初稿、請同事或前輩審閱 |
| **T-4 週（10 月中）** | 根據回饋修改、開始準備經費表 |
| **T-3 週（11 月初）** | 系統開放後開始線上作業、準備附件 |
| **T-2 週（11 月中）** | 完成線上填寫、上傳所有文件 |
| **T-1 週** | 最終校對、確認無遺漏 |
| **截止日** | 確認校內截止前送出、保留送出紀錄 |
