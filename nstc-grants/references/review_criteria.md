# 國科會專題研究計畫審查標準與流程

> **重要聲明**：本文件依據「國科會補助專題研究計畫作業要點」（112.11.6 修正）及國科會各處審查參考原則整理。確切規定以國科會最新公告為準。

---

## 一、審查方式

### 1.1 一般審查流程

```
計畫送件 → 學門分類 → 指派審查委員（2-3位）→ 書面審查 → 學門會議討論 → 核定結果
```

- 依「國科會研究計畫審查機制及審查委員遴選作業要點」規定辦理
- 審查作業期間：自申請案截止收件之次日起 **6 個月內**完成

### 1.2 整合型計畫審查

- 經審查不宜以整合型計畫補助者，子計畫如適合以個別型執行，國科會得**轉為個別型審查**
- 必要時得請計畫主持人至國科會**簡報**或由國科會至申請機構**實地訪查**

### 1.3 新進人員隨到隨審

- 從未申請國科會計畫者，得於起聘日/獲博士學位日/符合資格日起 **3 年內**，以隨到隨審方式申請
- 審查時程較大批計畫彈性

---

## 二、個別型研究計畫審查重點

國科會明訂個別型計畫審查五大重點：

### 2.1 計畫主持人研究表現與執行計畫能力

**審查面向**：
- 過去研究成果（論文發表、專利、技轉等）
- 執行過往計畫的成效
- 學術聲譽與研究能量
- 是否具備完成本計畫的專業能力

**應對策略**：
- 個人資料表中完整呈現近 5 年研究績效
- 強調過往計畫的具體產出成果
- 展示與本計畫相關的專業背景

### 2.2 計畫主題之重要性與創新性

**審查面向**：
- 研究議題的學術重要性
- 國際研究趨勢中的定位
- 創新程度（理論創新、方法創新、應用創新）
- 與現有研究的差異化

**應對策略**：
- 明確定義「gap」— 現有研究缺了什麼
- 說明本計畫如何填補此缺口
- 引用最新國際文獻佐證研究重要性
- 避免「微調型」研究，要有明確突破點

### 2.3 研究內容與方法之可行性

**審查面向**：
- 研究設計的邏輯嚴謹性
- 方法論的適切性與先進性
- 時程規劃的合理性
- 潛在風險與因應策略

**應對策略**：
- 提供具體研究步驟與時程甘特圖
- 說明為何選擇此方法而非其他方法
- 展示已有的前期成果或先導研究
- 納入替代方案（Plan B）

### 2.4 預期完成之項目與成果

**審查面向**：
- 預期成果的明確性與可衡量性
- 學術貢獻的具體內容
- 成果的應用潛力
- 對人才培育的貢獻

**應對策略**：
- 列出具體、可量化的預期成果
- 區分「確定可達成」與「挑戰性目標」
- 說明成果的學術與社會價值

### 2.5 經費與人力之合理性

**審查面向**：
- 經費需求是否與研究規模匹配
- 人力配置是否合理
- 各項預算是否有充分理由

**應對策略**：
- 每項經費都應有對應的研究活動說明
- 人力配置要明確說明各助理的具體工作內容
- 避免「灌水」或不合理的高額需求

## 三、整合型研究計畫額外審查項目

除個別型的五大重點外，整合型計畫另審查：

### 3.1 整合之必要性

| 審查面向 | 說明 |
|----------|------|
| 總體目標 | 是否有超越個別子計畫加總的整體目標 |
| 分工架構 | 各子計畫的角色定位是否清楚 |
| 子計畫相關性 | 各子計畫間是否有實質互動與依賴關係 |

### 3.2 人力配合度

| 審查面向 | 說明 |
|----------|------|
| 協調領導能力 | 總主持人是否有能力整合不同領域團隊 |
| 專業互補 | 團隊成員的專業是否互補而非重疊 |

### 3.3 資源整合程度

| 審查面向 | 說明 |
|----------|------|
| 儀器設備共用 | 是否規劃設備共享機制 |
| 經驗交流 | 是否有定期研討與交流機制 |
| 資料共享 | 是否有研究資料共享平台 |

### 3.4 申請機構配合度及綜合效益

| 審查面向 | 說明 |
|----------|------|
| 機構支持 | 執行機構是否提供足夠行政與資源支持 |
| 綜合效益 | 計畫整體對學術與社會的預期貢獻 |

## 四、各學門審查特色

不同學門的審查側重點有所差異：

| 學門 | 特別重視 |
|------|----------|
| **自然科學** | 理論深度、實驗設計嚴謹性、國際競爭力 |
| **工程技術** | 技術創新性、實用價值、產業應用潛力 |
| **生命科學** | 研究設計的科學性、倫理審查、實驗可重現性 |
| **人文社會** | 問題意識、理論架構、社會影響力、研究倫理 |

---

## 五、審查結果等級

國科會審查結果通常分為：

| 等級 | 意涵 | 後續處理 |
|------|------|----------|
| 通過 | 計畫獲得補助 | 依核定經費執行 |
| 修正後通過 | 需依審查意見修正 | 限期修正後核定 |
| 不通過 | 計畫未獲補助 | 可於次年重新申請 |

### 申覆機制

- 對審查結果有異議者，得依規定提出**申覆**
- 申覆期限依國科會公告辦理
- 申覆僅就程序瑕疵或事實認定錯誤提出，不得就學術判斷提出異議

## 六、提升通過率的實務建議

### 6.1 計畫書品質

- 文字精練、邏輯清晰，避免冗長
- 圖表善用，提升可讀性
- 遵守頁數限制（個別型 1 年期約 20 頁，多年期約 25 頁，整合型或年度經費 > 500 萬者約 40 頁）
- 中英文摘要與關鍵詞精確反映計畫核心

### 6.2 避免常見扣分項

| 扣分項 | 改善建議 |
|--------|----------|
| 研究動機空泛 | 具體指出要解決的科學問題 |
| 文獻回顧不足 | 涵蓋最新 2-3 年重要文獻 |
| 方法論模糊 | 提供具體步驟、參數、分析方法 |
| 創新性不足 | 明確指出與既有研究的差異 |
| 預期成果不具體 | 用量化指標呈現預期產出 |
| 經費不合理 | 每項經費對應具體研究活動 |

### 6.3 多年期計畫注意事項

- 多年期計畫順序應優先於新申請案
- 同一年度申請/執行 2 件以上計畫，國科會依件數**逐級從嚴**審查
- 應於計畫申請書內列明優先順序
- 期中各年執行期滿前 2 個月須繳交進度報告

---

## 七、利益衝突與學術倫理

### 申請時

- 計畫主持人需簽署利益衝突聲明
- 首次申請者須完成 **6 小時**學術倫理教育訓練並取得證明

### 涉及倫理審查的研究

| 研究類型 | 需取得之核准文件 |
|----------|----------------|
| 人體試驗/人體檢體 | 醫學倫理委員會或人體試驗委員會核准 |
| 基因重組實驗 | 生物實驗安全委員會核准 |
| 動物實驗 | 實驗動物管理委員會核准 |
| 人類胚胎/幹細胞 | 醫學倫理委員會核准 |
| 二級以上感染性材料 | 相關單位核准 |
| 人文社會科學涉及人的研究 | 研究倫理審查委員會核准 |

- 核准文件未能於申請時提交者，須先提交**已送審證明**，並於 **6 個月內**補齊
