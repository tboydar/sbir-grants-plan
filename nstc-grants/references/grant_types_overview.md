# 國科會各類研究計畫類型總覽

> ⚠️ **重要聲明**：本文件整合國科會官方公告與學術實務經驗，僅供撰寫輔助參考。確切規定以國科會最新公告為準。

---

## 一、專題研究計畫（最重要、最普遍）

### 1.1 個別型專題研究計畫

**性質**：國科會最核心的研究補助，支持學術人員進行獨立研究。

| 項目 | 內容 |
|------|------|
| **申請資格** | 公私立大學校院、研究機構之專任教師或研究人員 |
| **補助金額** | 依學門而異，一般 50-200 萬/年（特殊領域可更高） |
| **執行期限** | 1-5 年（一般 1-3 年，多年期需逐年審查） |
| **申請時間** | 每年約 7-10 月（依學門公告） |
| **審查方式** | 書面同儕審查（通常 2-3 位審查委員） |
| **核定率** | 約 40-60%（依學門、年度而異） |

**適合對象**：
- 有明確研究構想的教師/研究人員
- 已有初步成果想深入探索
- 需要經費支持研究助理與設備

### 1.2 新進人員研究計畫

**性質**：鼓勵新進學術人員建立獨立研究能量。

| 項目 | 內容 |
|------|------|
| **申請資格** | 取得博士學位 7 年內，首次申請或尚未獲得國科會計畫者 |
| **補助金額** | 一般 50-100 萬/年 |
| **執行期限** | 1-3 年 |
| **特殊優勢** | 核定率通常較個別型高（約 60-70%） |
| **審查重點** | 研究潛力、發展方向、獨立研究能力 |

**關鍵策略**：
- 強調個人學術養成與研究方向的一致性
- 展現「雖然年輕但已有初步成果」
- 明確說明研究方向的長期發展藍圖

### 1.3 整合型研究計畫

**性質**：鼓勵跨領域、跨機構合作研究。

| 項目 | 內容 |
|------|------|
| **組成** | 1 位總計畫主持人 + 3-5 位子計畫主持人 |
| **補助金額** | 總金額可達 500-2,000 萬/年 |
| **執行期限** | 通常 3 年（可續申） |
| **審查方式** | 書面審查 + 簡報審查 |
| **核定率** | 較低（約 30-40%） |

**關鍵策略**：
- 各子計畫間必須有明確的關聯性與互補性
- 總計畫需有整合機制（定期會議、共享資源、共同產出）
- 避免「拼裝車」問題 — 各做各的只是掛名整合

## 二、產學合作研究計畫

**性質**：促進學術研究成果應用於產業。

| 項目 | 內容 |
|------|------|
| **申請資格** | 學術機構研究人員，需有合作企業 |
| **補助金額** | 依規模，一般 100-500 萬/年 |
| **企業配合款** | 企業需出資配合（金額依計畫類型而定） |
| **執行期限** | 1-3 年 |
| **申請時間** | 通常每年 3-4 月公告 |

**子類型**：
- **一般型**：學術研究為主，產業應用為輔
- **應用型**：直接解決企業技術問題
- **開發型**：以技術移轉或商品化為目標

**與 SBIR 的區隔**：
- 產學合作計畫由「學術端」主導，SBIR 由「企業端」主導
- 產學合作仍重視學術產出（論文），SBIR 完全不看論文
- 產學合作的 PI 是教授，SBIR 的計畫主持人是企業負責人

## 三、大專學生研究計畫

**性質**：培養大學生研究興趣與能力。

| 項目 | 內容 |
|------|------|
| **申請資格** | 大學部學生（由指導教授提出申請） |
| **補助金額** | 研究獎助金約 6-8 萬/案 |
| **執行期間** | 暑假 7-8 月（約 8 週） |
| **申請時間** | 每年約 2-3 月 |
| **成果** | 需繳交研究成果報告、可投稿研究創作獎 |

## 四、國際合作研究計畫

**性質**：支持與國外學術機構的合作研究。

### 4.1 雙邊合作研究

| 項目 | 內容 |
|------|------|
| **合作國家** | 依國科會與各國簽署的合作協議 |
| **審查方式** | 雙邊同步審查 |
| **補助內容** | 研究經費 + 國際差旅 |

### 4.2 國際研究生/博士後交流

| 項目 | 內容 |
|------|------|
| **龍門計畫** | 博士候選人赴國外研究 |
| **千里馬計畫** | 博士後赴國外研究 |
| **補助內容** | 生活費 + 來回機票 + 研究費 |

### 4.3 舉辦國際學術研討會

| 項目 | 內容 |
|------|------|
| **補助額度** | 依規模，一般 20-80 萬 |
| **申請條件** | 需有國際參與者、學術品質要求 |

## 五、其他計畫類型

### 5.1 人文社會科學特有

- **人文社會科學研究中心計畫**
- **數位人文計畫**
- **學術研究倫理教育計畫**

### 5.2 特殊型計畫

- **傑出研究獎**：表彰優秀研究人員
- **特約研究員**：長期穩定支持傑出學者
- **學術攻頂計畫**：支持頂尖研究團隊
- **愛因斯坦計畫**：延攬海外傑出人才

### 5.3 重點領域計畫

國科會會依國家政策需求設定重點徵求領域，例如：
- AI 與半導體
- 淨零碳排
- 精準健康
- 太空科技
- 量子科技

## 計畫類型選擇建議

| 您的狀況 | 建議計畫類型 |
|----------|-------------|
| 新進教師，剛開始建立研究 | **新進人員計畫** |
| 有明確研究題目，獨立執行 | **個別型專題研究計畫** |
| 跨領域團隊，題目宏大 | **整合型研究計畫** |
| 研究成果有產業應用潛力 | **產學合作研究計畫** |
| 需要與國外團隊合作 | **國際合作研究計畫** |
| 指導大學部學生做專題 | **大專學生研究計畫** |
| 研究已很成熟，想衝國際頂尖 | **學術攻頂/傑出研究獎** |
