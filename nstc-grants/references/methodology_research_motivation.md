# 研究動機與目的撰寫方法論

> ⚠️ **重要聲明**：本方法論整合國科會審查實務與成功案例分析，僅供撰寫輔助參考，不保證採用即可通過審查。

---

## 審查委員在研究動機中看什麼？

根據國科會審查實務，審查委員會評估：

| 評估項目 | 審查重點 | 常見失敗原因 |
|----------|----------|--------------|
| **問題重要性** | 是否為領域內重要議題 | 題目太小、太邊緣 |
| **研究缺口** | 是否指出現有研究不足之處 | 缺乏文獻佐證、缺口不明確 |
| **原創性** | 是否有新的觀點或方法 | 只是重複前人工作 |
| **可行性** | 在提出的期程與經費內能否完成 | 野心太大、不切實際 |
| **學術貢獻** | 能否推進領域知識 | 貢獻不明確 |

---

## 三層論證法

成功的研究動機採用「**三層論證**」結構：

```
1️⃣ 宏觀背景（為何這個領域重要）
   → 引用產業趨勢/社會需求/科學前沿證明研究領域的重要性

2️⃣ 研究缺口（現有研究還缺什麼）
   → 透過文獻回顧指出尚未解決的關鍵問題

3️⃣ 本計畫定位（我們要怎麼填補缺口）
   → 清楚說明本計畫的研究目標與預期突破
```

---

## Step 1: 宏觀背景（為何重要）

### 1.1 建立研究領域的重要性

**目標**：讓審查委員（即使非該細分領域）也能理解這個研究議題的價值。

**資料來源**：
- 📊 國際權威期刊的 review/survey 論文
- 📊 國家科技政策白皮書（如國科會科技發展策略）
- 📊 國際重要報告（如 WHO、IPCC、World Economic Forum）
- 📊 產業報告（如研究主題有應用面）

**撰寫範例**：

❌ **避免空泛**：
> 人工智慧近年來發展迅速，應用範圍越來越廣。

✅ **具體且有說服力**：
> 根據 Nature（2024）的系統性回顧，大型語言模型在科學文獻探勘的準確率已從 2020 年的 62% 提升至 2024 年的 89%。然而，在特定領域（如材料科學）的術語辨識上，現有模型的 F1-score 仍僅有 0.71（Li et al., 2024），顯示領域適應性仍是亟待突破的瓶頸。

### 1.2 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 只講「很重要」但無具體佐證 | 引用具體數據或趨勢 |
| 過度鋪陳基礎知識 | 假設審查者有基本素養，聚焦核心 |
| 背景與研究主題脫節 | 每段背景都要能接到研究缺口 |
| 只用中文文獻 | 必須引用國際期刊文獻 |

## Step 2: 研究缺口（現有不足）

### 2.1 缺口類型

| 缺口類型 | 說明 | 範例 |
|----------|------|------|
| **理論缺口** | 現有理論無法解釋某現象 | 「現有的 XX 模型無法解釋在 YY 條件下的異常行為」 |
| **方法缺口** | 現有方法有明顯限制 | 「傳統 XX 方法在處理大規模資料時效率僅有 O(n³)」 |
| **實證缺口** | 缺乏某情境的實證資料 | 「目前尚無針對台灣 XX 族群的大規模研究」 |
| **應用缺口** | 理論好但實際應用有落差 | 「雖然 XX 理論已成熟，但實際部署時的 YY 問題尚未解決」 |

### 2.2 如何有效陳述缺口

**公式**：`雖然 [已知成果]，但 [具體不足]，因此 [需要什麼]`

**範例**：

> 雖然近年來深度學習在影像辨識的準確率已突破 95%（He et al., 2023），**但**在低光源環境下的辨識率仍大幅下降至 68%（Chen et al., 2024），且現有的資料增強方法對此類場景的改善有限（Kim et al., 2024）。**因此**，本計畫提出一種基於物理光學模型的合成資料生成方法，以系統性地解決低光源場景的訓練資料不足問題。

### 2.3 關鍵注意事項

- **不要否定所有前人研究**：態度是「站在巨人肩膀上」而非「前人都做錯了」
- **缺口要具體**：不是「研究不夠」，而是「缺乏 XX 條件下的 YY 研究」
- **缺口要可填補**：別提出你自己也解決不了的缺口

## Step 3: 本計畫定位（怎麼填補）

### 3.1 研究目的的撰寫

**目標**：明確、具體、可檢驗地說明本計畫要做什麼。

**結構建議**：

```
本計畫之主要研究目的如下：

目的一：[動詞] + [具體對象] + [預期達成標準]
   例：開發一套基於 XX 的 YY 系統，目標達到 ZZ% 的準確率

目的二：[動詞] + [具體對象] + [預期達成標準]
   例：建立 XX 理論模型，以解釋 YY 現象中的 ZZ 機制

目的三：[動詞] + [具體對象] + [預期達成標準]
   例：驗證 XX 方法在 YY 場景的適用性與限制
```

### 3.2 好的研究目的 vs 差的研究目的

| ❌ 模糊 | ✅ 具體 |
|---------|---------|
| 研究 AI 在醫療的應用 | 開發基於 Transformer 的胸腔 X 光異常偵測模型，目標 AUC > 0.95 |
| 探討永續能源議題 | 建立鈣鈦礦太陽能電池在高溫高濕環境的加速老化模型 |
| 分析社群媒體行為 | 利用 NLP 技術分析 2020-2025 年台灣政治性假訊息的傳播路徑與語言特徵 |

### 3.3 研究目的與研究問題的區別

| 維度 | 研究目的 (Objectives) | 研究問題 (Questions) |
|------|----------------------|---------------------|
| 形式 | 陳述句（我要做什麼） | 疑問句（答案是什麼） |
| 功能 | 引導執行方向 | 驅動知識探索 |
| 範例 | 「開發 XX 模型」 | 「XX 條件下，YY 會如何變化？」 |

## 完整範例：研究動機段落結構

```markdown
### 研究動機與目的

#### 一、研究背景
[2-3 段，建立領域重要性，引用關鍵文獻與趨勢數據]

#### 二、研究問題與缺口
[2-3 段，透過文獻回顧指出具體的研究缺口]

#### 三、研究目的
[條列式，明確列出 2-4 個具體研究目的]

#### 四、研究重要性與預期貢獻
[1-2 段，說明本研究的學術與應用價值]
```

## 自我檢核清單

- [ ] 研究背景是否有引用近 3 年內的關鍵文獻？
- [ ] 研究缺口是否具體明確（而非「研究不足」）？
- [ ] 研究目的是否可檢驗、可衡量？
- [ ] 背景 → 缺口 → 目的之間的邏輯是否連貫？
- [ ] 是否避免過度宣稱（如「全球首創」「完全解決」）？
- [ ] 是否適當引用自己的先前研究成果？
- [ ] 研究範圍是否在 1-3 年內可完成？
