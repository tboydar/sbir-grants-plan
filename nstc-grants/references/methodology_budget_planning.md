# 國科會計畫經費編列方法論

> ⚠️ **重要聲明**：本文件整合國科會官方規定與學術實務經驗，僅供撰寫輔助參考。確切金額、比例以國科會最新公告為準，每年度可能調整。

---

## 核心觀念：經費編列是研究可行性的具體展現

### 關鍵認知

經費編列不只是填數字，而是對審查委員說明：
1. 你清楚知道執行這項研究需要什麼資源
2. 你的研究規模與補助金額相符
3. 你能謹慎、負責地使用公款

**常見誤解**：「盡量多列一點，反正會被砍」→ 這個策略反而有害，不合理的預算會讓審查委員懷疑計畫可行性。

---

## 國科會經費類別架構

國科會計畫經費分為以下主要類別：

```
計畫總經費
├── 研究費（含人事費）
│   ├── 人事費
│   │   ├── 研究助理費（碩士級、博士級、學士級）
│   │   ├── 臨時工資
│   │   └── 博士後研究員（部分計畫）
│   └── 業務費
│       ├── 材料費（耗材）
│       ├── 國內差旅費
│       ├── 國外差旅費
│       ├── 雜支
│       └── 其他直接費用
├── 設備費
└── 管理費（由執行機構收取）
```

---

## 各項目詳細說明

### 一、人事費

#### 1.1 基本規則

- **主持人不得領取薪資**：PI（主持人）的薪資由所屬機構支付，國科會計畫不補助 PI 個人薪資
- **協同主持人**：同樣不得從計畫領取薪資（薪資由機構支付）
- **研究助理**：可由計畫支付薪資，依學歷級別有不同標準

#### 1.2 研究助理薪資標準（參考值）

| 學歷 | 每月薪資範圍（參考） | 備注 |
|------|------------------|------|
| 博士級研究助理 | 約 40,000-50,000 元 | 通常指博士候選人 |
| 碩士級研究助理 | 約 20,000-28,000 元 | 含在學碩士生 |
| 學士級研究助理 | 約 15,000-20,000 元 | 含在學大學生 |
| 臨時工（時薪制） | 依當年度基本工資計算 | 適用短期、兼職工作 |

> **重要**：以上為參考範圍，實際標準請依國科會當年度公告及所屬機構規定為準。部分機構有自訂標準。

#### 1.3 人事費計算方式

**全時研究助理（Full-time RA）**：
```
年度人事費 = 月薪 × 12 個月
（如跨年度執行，依計畫期程比例計算）
```

**半時研究助理（Part-time RA）**：
```
年度人事費 = 月薪 × 0.5 × 執行月數
```

**實務建議**：
- 碩士生 RA 通常列為 0.5 工時（另有學校獎學金）
- 博士生 RA 通常列為 1.0 工時
- 避免只列臨時工而無正式 RA，顯示研究支援不足

#### 1.4 博士後研究員

部分計畫（如整合型、主題計畫）可雇用博士後研究員：
- 薪資約 45,000-65,000 元/月（依機構規定）
- 需在計畫書中說明博後的研究職責
- 計畫期間需有正式雇用合約

---

### 二、業務費

業務費是執行研究所需的操作性費用，包含以下子項目：

#### 2.1 材料費（耗材費）

**涵蓋範圍**：
- 實驗耗材（試劑、玻璃器材、電子元件）
- 資料蒐集費（問卷印刷、田野調查材料）
- 文具、影印、書籍（不超過一定比例）
- 計算資源（雲端運算費用）

**撰寫原則**：
- 每項耗材應有合理用量估計
- 金額較大的單項耗材（如特殊試劑）需說明用途
- 可彙整為類別列表，不必明細到每根試管

**範例**：
```
材料費估計：
- 化學試劑（XX 實驗所需）：60,000 元
- 電子元件（PCB 打樣與測試）：40,000 元
- 問卷印製與郵寄：20,000 元
- 資料庫訂閱費：15,000 元
小計：135,000 元
```

#### 2.2 國內差旅費

**涵蓋範圍**：
- 國內研討會差旅
- 田野調查交通住宿
- 訪問合作機構

**常見上限**：每次出差有天數與金額上限，依國科會規定（通常與公務員差旅費相近）。

**撰寫建議**：
- 列出預計出席的國內研討會名稱（如 CIKM 台灣、各學會年會）
- 說明田野調查的地點與次數

#### 2.3 國外差旅費

**涵蓋範圍**：
- 國際研討會出席（含機票、住宿、研討會報名費）
- 國際學術訪問

**額度建議（參考）**：
| 目的地 | 機票（參考） | 住宿/天 | 報名費 |
|-------|------------|--------|-------|
| 歐美 | 40,000-60,000 | 3,000-5,000 | 15,000-30,000 |
| 亞洲 | 15,000-30,000 | 2,000-4,000 | 10,000-20,000 |

**重要限制**：
- 差旅費需有明確的學術目的（非觀光性質）
- 部分計畫有國外差旅次數限制（如每年不超過 2 次）
- 需於行前申請，行後報銷

#### 2.4 雜支

**涵蓋範圍**：
- 國際期刊版面費（Open Access 費用）
- 翻譯費、潤稿費
- 印刷裝訂費
- 電話費、郵資（少量）

**重要限制**：
- 雜支通常有上限，約為業務費的 10-20%（依機構規定）
- 不得用於購買與研究無關的個人物品

**Open Access 版面費特別說明**：
- 許多期刊的 OA 費用約 NT$30,000-150,000/篇
- 可在雜支中列入，但需說明是為提升論文能見度
- 部分機構另有 OA 補助，可雙軌申請

---

### 三、設備費

#### 3.1 定義

單價超過 **新台幣 10,000 元**的有形資產視為「設備」，需獨立列於設備費項目。

> 單價低於 10,000 元的物品，即使是電腦周邊、儀器配件，列入業務費（材料費）即可。

#### 3.2 設備費的規定

- **所有權**：以計畫補助購買的設備，所有權歸屬執行機構（非主持人個人）
- **維護責任**：計畫結束後，設備由機構統一管理維護
- **折舊問題**：計畫書中列設備費，是以「全額」計列，非折舊攤提

#### 3.3 設備費必須提供的資訊

申請設備費時，每項設備需說明：
1. 設備名稱與規格
2. 數量與單價
3. **研究用途**（這是最重要的，必須與計畫內容直接連結）
4. 是否為計畫執行的必要設備（而非便利性設備）

**範例**：
```
設備費：
1. 高效能工作站（含 GPU）× 1 台
   單價：120,000 元
   用途：本計畫需大量深度學習模型訓練，現有設備效能不足，
         預計每月訓練時間超過 500 小時，需專用 GPU 工作站。

2. XXX 精密量測儀器 × 1 台
   單價：350,000 元
   用途：[詳細說明]
```

#### 3.4 設備費常見問題

**問**：申請一台電腦（約 60,000 元）合理嗎？

**答**：需說明現有電腦無法支援研究需求。如果只是一般文書處理，不宜申請；如果需要特殊計算能力（如 GPU 訓練、大規模模擬），需詳細說明。

**問**：設備費是否有上限？

**答**：沒有硬性全額上限，但設備費過高（如超過計畫總費用的 50%）會引發審查關注，需特別加強說明必要性。

---

### 四、管理費（間接費用）

#### 4.1 管理費的性質

管理費（overhead / indirect costs）是執行機構向計畫收取的行政管理費用，用於支應：
- 水電、空調等共用設施
- 行政人員薪資（財務、人事）
- 機構基礎設施維護

#### 4.2 計算公式

```
管理費 = (計畫總經費 - 設備費) × 管理費率
```

**管理費率**：依各執行機構規定，通常為 **8-15%**（大學常為 10% 左右）。

**範例**：
```
計畫總經費：1,000,000 元
設備費：200,000 元
可計列管理費的基礎：800,000 元
管理費率：10%
管理費：80,000 元
```

> **注意**：管理費不是申請人「能用」的錢，而是機構留用。申請時應了解自己機構的費率，並在預算表中正確填寫。

## 多年期計畫的預算規劃

### 原則

- 各年度預算可以不同，視研究進程而定
- 第一年通常較高（設備採購、初步實驗）
- 後續年度可偏重人事費與差旅費
- 整體預算應呈現合理的研究進程邏輯

### 分年分配建議

| 費用類別 | 第一年 | 第二年 | 第三年 |
|---------|-------|-------|-------|
| 人事費 | 40% | 45% | 50% |
| 業務費 | 30% | 40% | 40% |
| 設備費 | 25% | 10% | 5% |
| 管理費 | 5% | 5% | 5% |

> 以上為示意比例，依研究性質調整。設備通常集中在第一年採購。

---

## 常見預算錯誤

### 錯誤一：人事費比例過低

**問題**：如果計畫只申請 20 萬但要養 3 名全職 RA，根本不夠用。

**改正**：人事費應占計畫費用的 40-60%（視研究類型），確保有足夠人力執行。

### 錯誤二：設備費無說明

**問題**：只列「XX 儀器：300,000 元」，沒有說明為何需要此設備。

**改正**：每項設備加一段說明，解釋現有資源不足及此設備在計畫中的具體用途。

### 錯誤三：雜支超出比例

**問題**：雜支填了 50 萬，超過業務費的 30%。

**改正**：若有大量 Open Access 費用，可在說明欄詳細列明，並請確認機構對雜支比例的限制。

### 錯誤四：忽略管理費

**問題**：預算表沒有填管理費，導致機構被迫從研究費中扣除。

**改正**：確認所屬機構的管理費率，在預算表中正確填入。

### 錯誤五：差旅費與研究計畫脫節

**問題**：計畫是純理論研究，但列了大量田野調查差旅費。

**改正**：差旅費必須與研究方法直接相關，並在計畫書中說明差旅目的。

---

## 預算編列的整體策略

### 由研究方法倒推預算

1. 先確定研究方法（需要什麼實驗、調查、計算）
2. 列出必要的人力（幾位 RA，全時/半時）
3. 列出必要的材料與設備
4. 估算差旅（幾次研討會，幾次田野）
5. 計算管理費
6. 加總，與補助額度上限對照

### 審查委員關注的預算信號

| 信號 | 解讀 |
|------|------|
| 人事費過低 | 研究人力不足，計畫恐難執行 |
| 設備費過高無說明 | 主持人想買設備而非做研究？ |
| 差旅費與計畫不符 | 計畫書與執行方式不一致 |
| 業務費細項空白 | 主持人不了解自己的研究需求 |
| 預算非常「整數」 | 顯示沒有認真估算，隨意填寫 |

## 快速檢核清單

提交前確認以下事項：

- [ ] 主持人與協同主持人未列薪資
- [ ] 研究助理薪資符合當年度標準
- [ ] 設備費每項均有研究用途說明
- [ ] 管理費已正確計算並填入
- [ ] 差旅費與計畫研究方法相符
- [ ] 雜支未超出機構規定比例
- [ ] 多年期計畫各年度預算有合理分配邏輯
- [ ] 預算總額在補助上限範圍內
