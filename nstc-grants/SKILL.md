# NSTC Grants Writing - 國科會專題研究計畫撰寫指南

## Overview

國科會（National Science and Technology Council，原科技部）是台灣最重要的學術研究補助機構，其「專題研究計畫」是學術界最核心的競爭型研究經費來源。本技能提供完整的計畫書撰寫指引，從研究動機到文獻探討，從研究方法到經費編列。

**核心原則：國科會計畫書必須同時展現學術原創性、研究方法嚴謹性、預期學術貢獻及研究團隊能量。** 與產業補助計畫（如 SBIR）不同，國科會強調的是「從假說到知識突破」的完整學術路徑。

## When to Use This Skill

此技能適用於：
- 規劃國科會專題研究計畫書架構
- 撰寫研究動機與目的
- 進行文獻探討與研究缺口分析
- 規劃研究方法與技術路徑
- 編列研究經費與人力配置
- 撰寫預期成果與學術貢獻
- 準備產學合作計畫書
- 新進人員計畫申請策略
- 整合型計畫規劃
- 避免常見的撰寫錯誤

## NSTC vs SBIR 關鍵差異

理解 NSTC 與 SBIR 的本質差異是成功的第一步：

| 維度 | NSTC (國科會) | SBIR (經濟部) |
|------|--------------|--------------|
| **目標對象** | 學術研究人員（大學/研究機構） | 中小企業 |
| **核心目標** | 科學突破 → 學術貢獻 | 技術創新 → 商業化 |
| **評審組成** | 學界專家為主 | 業界專家為主 |
| **評審重點** | 學術創新性、研究方法嚴謹性 | 市場潛力、產業效益 |
| **文獻探討** | **核心必要**，需深入完整 | 簡要，聚焦技術差異化 |
| **市場分析** | 不重要 | **核心必要** |
| **商業模式** | 不需要 | **核心必要** |
| **預期成果** | 論文、專利、學術影響力 | 產品、營收、技術移轉 |
| **團隊介紹** | 學術發表、研究經歷、指導紀錄 | 企業實績、產業經驗 |
| **經費性質** | 全額補助（無需自籌） | 需自籌 50% |
| **申請資格** | 需有學術機構聘書 | 需為中小企業 |
| **申請時程** | 每年固定（約 7-10 月） | 隨到隨審 / 定期徵件 |
| **執行期間** | 通常 1-3 年 | Phase 1: 6 個月 |

**關鍵心態轉換**：國科會審查委員問的是「這在學術上有什麼原創貢獻？研究方法是否嚴謹？能產出什麼學術影響力？」而非「這能不能賣？」

## 國科會計畫類型概覽

### 一、專題研究計畫（最主要）

**目標**：支持學術研究人員進行基礎或應用研究

| 項目 | 內容 |
|------|------|
| **補助上限** | 依學門而異，一般約 50-200 萬/年 |
| **期程** | 1-3 年（多年期需逐年審查） |
| **補助比例** | 全額補助 |
| **審查方式** | 書面審查（同儕審查） |
| **重點** | 學術原創性、研究方法、預期貢獻 |

**子類型**：
- **個別型計畫**：單一主持人獨立執行
- **整合型計畫**：多位主持人跨領域合作
- **新進人員計畫**：取得博士學位 7 年內的新進教師

### 二、產學合作研究計畫

**目標**：促進學術研究與產業需求結合

| 項目 | 內容 |
|------|------|
| **補助上限** | 依規模而異，一般 100-500 萬/年 |
| **期程** | 1-3 年 |
| **企業配合款** | 需企業出資配合 |
| **審查方式** | 書面審查 + 產業效益評估 |
| **重點** | 產學媒合、技術移轉可行性 |

### 三、大專學生研究計畫

**目標**：培養大學生研究能力

| 項目 | 內容 |
|------|------|
| **補助上限** | 約 6-8 萬/案 |
| **期程** | 暑假期間（約 2 個月） |
| **申請人** | 大學部學生（由指導教授提出） |
| **審查方式** | 書面審查 |

### 四、國際合作研究計畫

**目標**：促進國際學術交流與合作研究

| 項目 | 內容 |
|------|------|
| **形式** | 雙邊/多邊合作、國際研討會、短期訪問 |
| **合作對象** | 需有外國合作機構/研究者 |
| **審查方式** | 雙邊同步審查 |

## 計畫書核心章節

### 標準專題研究計畫書架構

1. **摘要**（中英文各 500 字以內）
2. **研究動機與目的**
3. **文獻探討與研究背景**
4. **研究方法與步驟**
5. **預期完成之工作項目及成果**
6. **經費需求**
7. **參考文獻**
8. **主持人學經歷及研究成果**

## 經費編列架構

### 主要經費科目

| 科目 | 說明 | 注意事項 |
|------|------|----------|
| **人事費** | 研究助理薪資、臨時工資 | 主持人不得支薪，助理依學歷分級 |
| **業務費** | 材料、差旅、雜支等 | 各細項有比例上限 |
| **設備費** | 單價 1 萬元以上的儀器設備 | 需說明必要性，不得重複購置 |
| **管理費** | 學校行政管理費 | 依補助金額一定比例計算 |

### 經費編列比例建議

| 計畫類型 | 人事費 | 業務費 | 設備費 | 管理費 |
|----------|--------|--------|--------|--------|
| **理工類** | 30-40% | 25-35% | 15-25% | 依規定 |
| **人文社科** | 40-55% | 30-40% | 5-15% | 依規定 |
| **跨領域** | 35-45% | 25-35% | 10-20% | 依規定 |

## 審查標準

### 主要評審項目（一般專題研究計畫）

| 評審項目 | 權重 | 說明 |
|----------|------|------|
| **研究主題之重要性** | ~20% | 是否具學術價值與前瞻性 |
| **研究內容與方法** | ~30% | 創新性、方法論嚴謹度 |
| **預期成果與貢獻** | ~20% | 學術影響力、具體產出 |
| **主持人研究能力** | ~20% | 過往研究成果、執行力 |
| **經費合理性** | ~10% | 編列是否合理、必要 |
