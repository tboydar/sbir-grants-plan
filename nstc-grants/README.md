## 與 SBIR 的差異

如果您同時考慮申請 SBIR，請參閱 [NSTC vs SBIR 差異比較](references/nstc_vs_sbir.md)。

**一句話摘要**：國科會看的是「學術原創性」，SBIR 看的是「商業化潛力」。

## 重要時程提醒（依 115 年度/2026 年實際時程更新）

| 時間點 | 事項 |
|--------|------|
| **每年 10-11 月** | 國科會公告年度專題研究計畫徵求函 |
| **每年 11 月初** | 線上申請系統開放（115 年度為 114/11/4） |
| **每年 12 月底-1 月初** | 大批計畫校內截止（以各校研發處規定為準） |
| **每年 1 月上旬** | 研發處函送國科會 |
| **每年 6-7 月** | 審查結果公告 |
| **每年 8 月 1 日** | 新年度計畫開始執行 |
| **隨到隨審** | 新進人員計畫（3 年內未申請者） |
| **不定期** | 國際合作計畫（依合作國時程） |

> 確切日期每年微調，請以國科會官網公告為準。
> 詳細時程請見 [申請時程與流程](references/application_timeline.md)

## 2025-2026 最新政策更新

- **博士生費用新制**：自 114 年起，博士生兼任人員費用須於計畫申請時即編列，不再事後追加
- **青穗學者計畫**：原「2030 跨世代年輕學者方案」轉型，每年補助不超過 NT$800 萬，至多 4 年（可續 2 期共 8 年）
- **優秀年輕學者計畫**：45 歲以下，每年不超過 NT$500 萬，至多 4 年
- **國際合作擴大**：115-116 年度持續擴大雙/多邊合作計畫（德、法、日、加、捷、奧、比、瑞典等）
- **前瞻產學合作**：大型前瞻技術產學合作計畫企業年度貢獻可達數百萬至千萬元，國科會補助上限提高
