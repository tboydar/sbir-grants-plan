## 與 SBIR 的差異

如果您同時考慮申請 SBIR，請參閱 [NSTC vs SBIR 差異比較](references/nstc_vs_sbir.md)。

**一句話摘要**：國科會看的是「學術原創性」，SBIR 看的是「商業化潛力」。

## 重要時程提醒

| 時間點 | 事項 |
|--------|------|
| **每年 7-8 月** | 國科會公告下年度專題研究計畫徵求 |
| **每年 9-10 月** | 專題研究計畫申請截止（依學門而異） |
| **每年 1-2 月** | 審查結果公告 |
| **每年 3 月** | 產學合作計畫徵求公告 |
| **每年 8 月 1 日** | 新年度計畫開始執行 |

> ⚠️ 確切日期每年微調，請以國科會官網公告為準。
